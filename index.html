真正的第一篇记叙

搭建博客遇到的问题

   这次搭建博客遇到好多事，虽然是按照视频一步步弄得，但是首先就遇到一个电脑提示没有git命令的问题

还没学JavaScript，当然也没听过nodejs，git就只能硬着头皮按照视频一步步来。
当然也看了好几个视频。

整理一下看的coedsheep的搭建博客视频

准备工作：
    下载nodejs，git。（都是在官网下载）
    命令行npm install -g hexo-cli(全局安装hexo框架)
    cnpm install --save hexo-deployer-git(安装git部署插件)
开始：
    创建文件夹(起名)，进入该路径。
    hexo init   //初始化博客
    hexo s     //本地启动博客
    hexo n    //生成新文章
     hexo clean  //清理
    hexo g      //生成静态文件，我也不了解看着这个网站有讲解  "https://hexo.io/zh-cn/docs/commands.html"
配置-config.yml
    目前具体配置最下面的那几段。
    type: git
    repo: url
    branch: master
部署：
    hexo d
    //我没实验，只是听他说我感觉url是自己的网站且能访问就行，他具体讲的是github上创建的仓库，注意仓库名要取   用户名.github.io
    对了输入账号密码还有点问题，好像是需要事先打开github进入仓库，然后命令行部署时会弹出提示
